Cloudiness

There appears to be no correlation between Latitude and Cloudiness. Looking at the output of data on the above scatterplot you can see that common cloudiness levels are 0% cloud cover, approximately 78% cloud cover, and approximately 90% cloud cover. These findings seem to be consistent across all latitudes.
